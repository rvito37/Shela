# Shela — AI-портал для израильских женщин

## Видение

Ивритский контент-портал в стиле онлайн-журнала. Каждую рубрику ведёт своя AI-колумнистка с уникальным голосом и стилем повествования, вдохновлённым реальными израильскими журналистками.

**Целевая аудитория:** Израильские женщины 25-55 — мамы (имахот), карьеристки, девушки интересующиеся духовностью и стилем жизни.

**Фишка:** Не блог, а журнал. 7 AI-персон, каждая со своим характером, стилем письма и рубрикой. Как будто читаешь настоящий журнал с разными колумнистками.

---

## AI-персоны (колумнистки)

У каждой — своё имя, аватар, биография, стиль и зона экспертизы. Стили вдохновлены реальными израильскими журналистками и трендами.

### 1. Noa (נועה) — «Острый глаз»
- **Вдохновлена:** Ирит Линур (сатира, острый юмор)
- **Рубрика:** Культура и общество (תרבות וחברה)
- **Тон:** Острый, прямой, юмор с подтекстом, феминистский взгляд
- **Стиль:** Короткие хлёсткие фразы, сленг, смелые мнения
- **Темы:** Гендер, социальные тренды, поп-культура, израильское общество

### 2. Tamar (תמר) — «Мудрая подруга»
- **Вдохновлена:** Классические израильские колонки советов
- **Рубрика:** Отношения и жизнь (זוגיות וחיים)
- **Тон:** Тёплый, эмпатичный, сестринский
- **Стиль:** Личные истории, практические советы, разговорный
- **Темы:** Отношения, свидания, семья, эмоциональное здоровье, селф-кеа

### 3. Yael (יעל) — «Современная Ima»
- **Вдохновлена:** Това Лей и израильские мама-блогерши
- **Рубрика:** Материнство и семья (אמהות ומשפחה)
- **Тон:** Реалистичный, самоироничный юмор, честность
- **Стиль:** Личные эссе, листиклы, «давай начистоту»
- **Темы:** Воспитание, work-life balance, школа, быт, мамская жизнь в Израиле

### 4. Shira (שירה) — «Духовный путеводитель»
- **Вдохновлена:** Сиван Рахав Меир (культурно-религиозная журналистика)
- **Рубрика:** Иудаизм и духовность (יהדות ורוחניות)
- **Тон:** Вдумчивый, мост между светским и религиозным, тёплый но глубокий
- **Стиль:** Тора + современная жизнь, доступная глубина
- **Темы:** Шабат, праздники, еврейский майндфулнесс, каббала, духовный рост

### 5. Dana (דנה) — «Трендсеттер»
- **Вдохновлена:** Анна Зак и израильские бьюти/фэшн инфлюенсеры
- **Рубрика:** Красота и мода (יופי ואופנה)
- **Тон:** Энергичный, уверенный, трендовый, доступная аспирация
- **Стиль:** Визуально-ориентированный, обзоры продуктов, тренд-репорты
- **Темы:** Мода, скинкер, бьюти-продукты, израильские дизайнеры, велнес-бьюти

### 6. Michal (מיכל) — «Бизнес-леди»
- **Вдохновлена:** Став Шафир / Люси Ахариш (смелый голос)
- **Рубрика:** Карьера и деньги (קריירה וכסף)
- **Тон:** Прямой, empowering, без воды, мотивационный
- **Стиль:** Экшн-советы, истории успеха, данные и факты
- **Темы:** Карьера, зарплата, предпринимательство, финграмотность, равенство

### 7. Avigail (אביגיל) — «Поэт кухни»
- **Рубрика:** Еда и дом (אוכל ובית)
- **Тон:** Тёплый, чувственный, ностальгичный но современный
- **Стиль:** Рецепты-истории, кулинарные эссе, сезонный контент
- **Темы:** Израильские/ближневосточные рецепты, ЗОЖ-еда, шабатний стол, фуд-тренды

## Контент-архитектура

### Типы контента
1. **Статьи** (מאמרים) — 800-1500 слов, основной формат
2. **Коротко** (בקצרה) — 200-400 слов, реакции на тренды
3. **Списки** (רשימות) — «7 способов...», «Топ-10...»
4. **Вопрос-ответ** (שאלות ותשובות) — Читательские вопросы
5. **Еженедельные колонки** (טורים שבועיים) — Регулярные рубрики

### Пайплайн генерации контента
1. **Выбор темы** — Тренды из израильских СМИ + evergreen контент-календарь
2. **AI-генерация** — Каждая персона со своим system prompt и стилевыми инструкциями
3. **Проверка качества** — Автоматические проверки + опциональный ручной ревью
4. **SEO-оптимизация** — Ивритские ключевики, мета-теги, structured data
5. **Публикация** — По расписанию или сразу через админку

### Расписание публикаций
| Рубрика | Персона | Частота |
|---------|---------|---------|
| Культура и общество | Noa | 3 раза/неделя |
| Отношения и жизнь | Tamar | 2 раза/неделя |
| Материнство | Yael | 3 раза/неделя |
| Иудаизм и духовность | Shira | 2 раза/неделя + праздники |
| Красота и мода | Dana | 3 раза/неделя |
| Карьера и деньги | Michal | 2 раза/неделя |
| Еда и дом | Avigail | 2 раза/неделя + шабат |

**Итого: ~17 публикаций в неделю**

## Технический стек

### Основной стек
| Компонент | Технология |
|-----------|-----------|
| Фреймворк | Next.js 15+ (App Router) + TypeScript |
| Стилизация | Tailwind CSS v4 + `tailwindcss-flip` (RTL) |
| Хостинг | Vercel |
| База данных | Supabase (PostgreSQL) + RLS-политики |
| Аутентификация | `@supabase/ssr` (cookie-based, для админки) |
| AI-модели | Claude Sonnet 4.6 (основная) + GPT-4o (разнообразие) + Gemini (картинки) |
| i18n / RTL | `next-intl` v4.0+ (~2KB, нативный App Router) |
| Картинки | Supabase Storage + Image Transformations + Unsplash API + AI-генерация |
| Аналитика | PostHog (1M events/месяц бесплатно) + Vercel Analytics + Google Search Console |
| Рассылки | Resend + React Email |
| SEO | Next.js Metadata API + JSON-LD + динамический sitemap |
| Шедулинг | `pg_cron` + Vercel Cron Jobs |

### Схема базы данных (Supabase)

```sql
-- Персоны (AI-колумнистки)
personas
  id              uuid PK
  name            text          -- Ивритское имя (נועה)
  name_en         text          -- Английское имя (Noa)
  slug            text UNIQUE
  bio             text          -- Биография на иврите
  avatar_url      text
  style_prompt    text          -- System prompt для AI-генерации
  ai_model        text          -- Какую модель использовать
  rubric_name     text          -- Название рубрики на иврите
  rubric_slug     text
  is_active       boolean

-- Статьи
articles
  id              uuid PK
  persona_id      uuid FK -> personas
  title           text
  slug            text UNIQUE
  excerpt         text
  content         text          -- Markdown
  content_type    enum (article, quick_take, list, qa, column)
  cover_image_url text
  tags            text[]
  seo_title       text
  seo_description text
  status          enum (draft, scheduled, published, archived)
  published_at    timestamptz
  scheduled_for   timestamptz
  ai_model_used   text
  generation_prompt text
  word_count      int
  reading_time_minutes int
  view_count      int DEFAULT 0
  created_at      timestamptz
  updated_at      timestamptz

-- Теги
tags
  id              uuid PK
  name            text          -- На иврите
  slug            text UNIQUE

-- Связка статьи-теги
article_tags
  article_id      uuid FK
  tag_id          uuid FK

-- Подписчики
subscribers
  id              uuid PK
  email           text UNIQUE
  name            text
  subscribed_rubrics text[]     -- На какие рубрики подписан
  is_active       boolean
  created_at      timestamptz

-- Расписание контента
content_schedule
  id              uuid PK
  persona_id      uuid FK
  topic           text
  content_type    text
  scheduled_for   timestamptz
  status          enum (pending, generating, generated, published, failed)
  article_id      uuid FK NULLABLE
```

### Иврит / RTL
- Корневой `<html dir="rtl" lang="he">`
- `next-intl` v4.0+ для локализации (нативный для App Router, Server Components)
- Tailwind CSS logical properties (`ms-`, `me-`, `ps-`, `pe-`) + `tailwindcss-flip`
- Шрифты: **Heebo** (чистый, современный) или **Assistant** (элегантный)
- Навигация справа налево
- Тестировать с носителями иврита

### SEO-стратегия (критично для AI-контента)
- Google в Dec 2025 начал жёстко бить по массовому AI-контенту (87% негативное влияние)
- **Формула успеха:** AI-черновик + проверка качества + E-E-A-T сигналы + уникальные израильские инсайты
- Каждая персона = «эксперт-автор» с биографией, фото, постоянным голосом (E-E-A-T)
- Next.js Metadata API для динамических мета-тегов
- JSON-LD structured data (Article, Person/Author, BreadcrumbList)
- Динамический sitemap через `sitemap.ts`
- Ивритский keyword research по каждой рубрике

## Ключевые решения

1. **Журнал, а не блог** — Грид-лейаут с featured-статьями, аватары персон, секции рубрик
2. **Персона-first навигация** — Можно подписаться на конкретную «колумнистку»
3. **Mobile-first** — 70%+ израильтянок сидят с телефона (Instagram/TikTok привычки)
4. **Быстрая загрузка** — ISR для страниц статей
5. **Hebrew-native** — Не переведённый сайт, а построенный для иврита с нуля
6. **PostHog аналитика** — 1M events/месяц бесплатно, без проблем с GDPR

## Исследование: что сейчас в тренде

### Популярные порталы
- **Ynet** (+ La'Isha), **Mako**, **Walla!**, **Globes** — основные площадки
- La'Isha (לאשה) — флагманский женский журнал с 1947 года

### Горячие темы для израильских женщин (2025-2026)
- **Мода:** Animal prints, пудровый розовый, оливковый, минимализм + пустынные тона
- **Бьюти:** Longevity mindset, clean beauty, рефил-упаковка, персонализация
- **Здоровье:** Энергия > похудение (29%), gut health, гормоны, пробиотики
- **Материнство:** Баланс между «идеальная мама» и карьера, сингл-мамы by choice
- **Духовность:** Пост-7-октября — рост интереса к Шабату, молитве, еврейской медитации
- **Соцсети:** Instagram доминирует (88% в Тель-Авиве), TikTok растёт (54%), Telegram взрывной рост (70%)

### Стили израильских журналисток
- **Прямота** — Subtlety = уклончивость. Израильтянки ценят прямой разговор
- **Юмор** — Сильный, чёрный, политизированный, наблюдательный
- **Личный голос** — Колонка = бренд автора, не издания
- **Позиция** — Нейтральность не работает. Лучшие колумнистки занимают сторону
- **Аутентичность** — Личные истории, уязвимость, не корпоративный тон
